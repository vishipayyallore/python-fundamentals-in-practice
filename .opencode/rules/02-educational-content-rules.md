# Educational Content Rules

Canonical detail: `.cursor/rules/01_educational-content-rules.mdc`.

- Create transformative, original teaching material; never copy or lightly paraphrase third-party tutorials.
- Keep sessions realistic for about 30 minutes; split large topics instead of trimming away learning value.
- Show the why behind code choices, not only syntax.
- Pair concepts with small runnable practice scripts where useful.
- Use Mermaid-first diagrams with ASCII fallback for conceptual flows.
- Preserve beginner vocabulary and the existing curriculum sequence.
