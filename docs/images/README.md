# Session images

Educational screenshots and diagrams for session documentation live under `docs/images/S{session}/`.

## Required assets (optional — S1 has ASCII/Mermaid fallbacks)

| Session | File | Referenced in | Fallback |
| --- | --- | --- | --- |
| S1 | `S1/Py_Source_ByteCode.PNG` | `docs/sessions/L1/S1.md` | ASCII + Mermaid in session doc |
| S1 | `S1/Help_V1.PNG` | `docs/sessions/L1/S1.md` | Sample `help()` output in session doc |
| S1 | `S1/Help_V2.PNG` | `docs/sessions/L1/S1.md` | Sample `help(print)` output in session doc |

These PNG files are instructor-local assets. They are not committed to the public repository clone. Restore them from the instructor asset store when you want screenshot-based teaching; the session doc works without them.

## Naming convention

- Format: `{descriptive_name}.PNG` or `.png`
- Location: `docs/images/S{session}/`

See `docs/02_RepositoryStructure.md` for the full image reference pattern.
