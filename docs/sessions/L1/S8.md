# Session 8: Lists & Loops

**Duration:** 30 minutes  
**Type:** 📚 Knowledge  
**Level:** Noob → Nerd

---

## 🎯 Session mission: store many values in one name

Variables hold one value; **lists** hold an ordered sequence you can grow, shrink, and loop over. This is the collection you'll use in almost every real script.

### What you'll learn

- Create lists and read items by **index** (including `-1` for the last item).
- **Slice** sub-ranges without copying the whole story by hand.
- Change lists with **methods** — `append`, `remove`, `insert`, `pop`, `sort`, `reverse`.
- Combine **`for` loops** with `enumerate()` for numbered output.
- Build a **task manager** that accepts commands in a loop.

## 🧰 Troubleshooting

1. **IndexError: list index out of range**  
   Valid indices are `0` through `len(items) - 1`. Index `-1` is the last item only.

2. **ValueError on remove()**  
   `remove(x)` needs an **equal value** present in the list — use `pop(index)` to delete by position.

3. **Infinite task-manager loop**  
   Ensure `quit` calls `break` and that you update any counter inside `while` loops.
